Yarhl: Yet Another ROM Hacking Library

Yarhl is a library for ROM Hacking and fan-translation projects. It provides a virtual file system, file format and format conversion features and plugin support. It's built in C# / .NET and works in Windows, Linux and Mac OS X.

Install

Stable releases are available from nuget.org:

  • Yarhl
  • Yarhl.Media

Alpha releases can be found in this Azure DevOps package repository. To use an alpha release, create a file nuget.config in the same directory of your solution (.sln) file with the following content:

<?xml version="1.0" encoding="utf-8"?>
<configuration>
  <packageSources>
    <add key="Yarhl-Alpha" value="https://pkgs.dev.azure.com/SceneGate/Yarhl/_packaging/preview%40Local/nuget/v3/index.json" />
  </packageSources>
</configuration>

Build

Since the unit tests are validated against .NET Core and .NET Framework or Mono both runtime must be installed on the machine. Then run:

dotnet tool restore
dotnet cake
